Data files from 40 participants, located in subject folders 001-0402. Archival copies of the EEGLAB/ERPLAB scripts and accessory files used for analyzing the data. The updatedversions of the scripts are hosted on GitHub (https://github.com/lucklab/ERP_CORE)3. The event code scheme for the P3 active visual oddball task ("task-P3_events.json")4. Demographic information for all 40 participants ("participants.tsv &.json")~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~                                              Contact Us~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~To download the EEGLAB/ERPLAB data analysis scripts and accessory files used for analyzing the data, as wellas the processed EEG and ERP data, visit https://osf.io/thsqg/To download the latest versions of the scripts or to report bugs, visit https://github.com/lucklab/ERP_COREFor other questions about the ERP CORE, contact us at erpbootcamp@gmail.com. Additional ERP resources are available on our website (https://erpinfo.org).
